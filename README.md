```bash
git clone https://github.com/jonathanmagambo/otterlang.git
cd otterlang
nix develop
cargo build --release
./target/release/otter run hello.ot
```

See the [Getting Started Guide](docs/GETTING_STARTED.md) for detailed instructions.
